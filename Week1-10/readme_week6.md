## วิธีทดลอง
1. แก้ไข config.py (HOST, BASE_PORT, NEIGHBORS, FORWARD_PROBABILITY, TTL, BUFFER_SIZE)
2. เปิด node หลาย Terminal: python node.py (แต่ละ node ใช้ BASE_PORT ต่างกัน)
3. สังเกตการส่งต่อ message ระหว่าง node
